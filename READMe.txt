Hey There!
This is a memory game built using Html,Css and JavaScript(DOM events).

How to Setup:
1. Open index.html file.
2. Make the index.html live by clicking Golive button.
3. Make sure all the files (like script.js and style.css) are present in the same folder where index.html also exists.

Steps to Play:
1. Click on a card to flip it.
2. Flip another card to find its match.
3. If the cards match, they remain face-up.
4. If they do not match, they flip back after a short delay.
5. The game ends when all pairs are found.
